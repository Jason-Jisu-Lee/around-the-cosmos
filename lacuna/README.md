# Lacuna

A calm, celestial idle game. Planets orbit a sun and pay stardust. Tap the orbit to harvest early. Catch comets for windfalls. Prestige by collapsing the star. Win by igniting a supernova.

## Run it

```bash
npm install
npm run dev
```

Then open `http://localhost:3000` in your browser.

Or just open `index.html` directly — no build step needed.

## Debug mode

Add `?debug` to the URL for cheat buttons (inject dust/remnants, speed multiplier, force comets).
